# Scripting

**Scripting** is an iOS app that brings the power of JavaScript and TypeScript to your device. Build scripts, automations, widgets, Live Activities, notifications, and integrations with native iOS capabilities.

> 在 iPhone 与 iPad 上，使用 JavaScript / TypeScript 构建脚本、自动化、组件与原生体验。

## Explore

- 📚 **[Documentation](https://scriptingapp.github.io)** — Guides, API references, and examples.
- 🧩 **[Community Scripts](https://github.com/ScriptingApp/Community-Scripts)** — Scripts shared by the Scripting community.
- 🛠️ **[Skills](https://github.com/ScriptingApp/skills)** — Reusable skills that run in Scripting.
- 📦 **[Package Manager](https://github.com/ScriptingApp/Package-Manager)** — Install npm packages and download bundled JavaScript files in Scripting.
