# Scripting App

> Your AI agent and automation workspace on iPhone and iPad.

[![Download on the App Store](https://img.shields.io/badge/Download_on_the-App_Store-0D96F6?logo=apple&logoColor=white)](https://apps.apple.com/us/app/scripting/id6479691128)
[![Documentation](https://img.shields.io/badge/Documentation-scriptingapp.github.io-24292f?logo=readthedocs&logoColor=white)](https://scriptingapp.github.io)

**Scripting App** is an AI agent app for iOS that turns natural-language requests into useful work. It combines AI agents with JavaScript and TypeScript, native iOS APIs, extensible tools, and a powerful development environment—so you can create, run, and share automations directly on your device.

## 🤖 AI Agents That Get Things Done

Ask an agent in natural language, then let it plan, create, and execute work with your approval.

- **Build with AI** — Generate, inspect, edit, diagnose, and run JavaScript and TypeScript projects.
- **Use connected tools** — Extend agents with MCP servers and reusable skills for GitHub, Gmail, maps, calendars, and other services.
- **Work with files and media** — Create and manage project files; generate documents, images, audio, and video workflows.
- **Automate real tasks** — Research information, organize data, create content, manage repositories, and orchestrate multi-step workflows.
- **Stay in control** — Sensitive or device-affecting operations require your approval, while credentials can remain securely stored in Keychain.

## ✨ Native iOS Superpowers

Scripting App connects AI and code to the capabilities of iOS:

- Create iOS-style pages, Home Screen widgets, Live Activities, and rich notifications.
- Use Calendar, Reminders, Health, Location, Photos, speech, alarms, and other system APIs.
- Expose scripts through Shortcuts, the Share Sheet, App Intents, and interactive widgets.
- Build Safari userscripts and custom browser functionality.
- Install and use npm packages directly in Scripting App projects.
