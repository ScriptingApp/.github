# Scripting App

> Your AI agent and automation workspace on iPhone and iPad.

[![Download on the App Store](https://img.shields.io/badge/Download_on_the-App_Store-0D96F6?logo=apple&logoColor=white)](https://apps.apple.com/us/app/scripting/id6479691128)
[![Documentation](https://img.shields.io/badge/Docs-scriptingapp.github.io-24292f?logo=readthedocs&logoColor=white)](https://scriptingapp.github.io)

**Scripting App** is an AI agent app for iOS that turns natural-language requests into useful work. It combines AI agents with JavaScript and TypeScript, native iOS APIs, extensible tools, and a powerful on-device development environment—so you can create, run, and share automations directly from your iPhone or iPad.

## 🤖 AI Agents That Get Things Done

Ask an agent in natural language, then let it plan, create, and execute work with your approval. Scripting App gives agents the tools to move beyond chat and interact with the services, files, scripts, and iOS capabilities that power your workflows.

- **Build with AI** — Generate, inspect, edit, diagnose, and run JavaScript and TypeScript projects from a conversation.
- **Use connected tools** — Extend agents with MCP servers and reusable skills for services such as GitHub, Gmail, maps, calendars, and more.
- **Work with files and media** — Create and manage project files; generate documents, images, audio, and video workflows.
- **Automate real tasks** — Research information, organize data, create content, manage repositories, and orchestrate multi-step workflows.
- **Stay in control** — Sensitive or device-affecting operations require your approval, while credentials can remain securely stored in Keychain.

## ✨ Native iOS Superpowers

Scripting App connects AI and code to the capabilities of iOS:

- **Automations & utilities** — Build personal tools, workflows, and integrations with web services.
- **Native interfaces** — Create iOS-style pages, Home Screen widgets, Live Activities, and rich notifications.
- **System APIs** — Work with Calendar, Reminders, Health, Location, Photos, notifications, speech, alarms, and more.
- **Shortcuts & App Intents** — Expose scripts through the Shortcuts app, Share Sheet, and interactive widget controls.
- **Safari extensions** — Build browser userscripts and add custom Safari functionality.
- **npm packages** — Install and use npm packages directly in Scripting App projects.
